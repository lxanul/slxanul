# lxanul. - Profesjonalne Portfolio Web Developera

Nowoczesna, ultraszybka i responsywna strona typu wizytówka/portfolio dla **lxanul.**, prezentująca ofertę premium w zakresie tworzenia stron internetowych. Projekt ukierunkowany na doskonałą wydajność (Core Web Vitals) i nieszablonowy, elegancki minimalizm.

## 🚀 Funkcjonalności
- **W 100% Responsywna (RWD):** Kod zoptymalizowany pod kątem prawidłowego wyświetlania na smartfonach, tabletach i komputerach PC.
- **Tryb Ciemny/Jasny (Dark Mode):** Przełącznik motywu z płynnym przesyłaniem zmiennych w CSS, zapisujący wybór użytkownika w klastrze przeglądarki (`localStorage`).
- **Płynne Animacje (Scroll Reveal):** Strona dynamicznie odkrywa swój układ podczas przewijania (wykorzystanie `IntersectionObserver` zamiast zasobożernych bibliotek).
- **Aktywny Formularz Kontaktowy:** Implementacja technologii Web3Forms z obsługą żądań Fetch API, wbudowanymi alertami bezpieczeństwa i ładowaniem bez przeładowania strony.
- **Agresywne SEO & Optymalizacja:** Semantyczny kod HTML, pliki `robots.txt`, gotowa sitemap, tagi w formacie Open Graph / Twitter Card i wsparcie dla wektorowych ikon wysokiej rozdzielczości (`.svg`).

## 🛠️ Zastosowane Technologie
- **HTML5 & CSS3** - Pisane całkowicie w standardzie Vanilla, bez używania wielkich i obciążających frameworków, utrzymywane w ryzach lekkiego kodu.
- **JavaScript (Vanilla JS)** - Ograniczony do absolutnego i dynamicznego minimum manipulacji w drzewie DOM.

## 📁 Struktura Projektu
- `index.html` - Główny hub biznesowy / Landing Page.
- `polityka-prywatnosci.html` - Minimalistyczna podstrona prawna.
- `styles.css` - Główne style CSS, reguły motywów dark mode, breakpoints mobilne.
- `favicon.svg` / `favicon.png` - Autorskie, minimalistyczne materiały marki przygotowane nawet pod ekrany Apple Retina.

## ⚙️ Wdrożenie (Deploy)
Strona jest w formacie czystym (`Static Site`). Aby opublikować ją w internecie, z łatwością skopiuj wszystkie pliki z głównego folderu i zrzuć je na darmowy hosting (m.in.: **GitHub Pages, Vercel, Netlify**) lub skorzystaj z oprogramowania FTP dowolnego standardowego serwera (np. cyberfolks, hostinger).
